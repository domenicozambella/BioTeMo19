Il materiale didattico di questo corso è archiviato in formato grezzo in questa repository 
gitlab.com/domenicozambella/BioTeMo19
Per utilizzarlo al meglio, bisogna disporre dell'applicazione Jupyter che può essere installata su un quialsiasi computer e si utilizza con un qualsiasi browser. Per scaricare la repository bisogna aprire un terminale (qualsiasi terminale, ce n'è uno all'interno di Jupyter) e dare il comando 
git clone https://gitlab.com/domenicozambella/BioTeMo19.git
Per aggiornare la repository bisogna, di nuovo con un termilale, spostarsi nella directory BioTeMo19 e dare il comando 
git fetch origin; git reset --hard origin/master
Può essere più semplice e PREFERIBILE utilizzare il server Jupyter di ateneo. Agli studenti del primo anno verranno (o sono già state a qualcuno) inviate per email all'indirizzo istituzionale le credenziali per accedere al server. La procedura per scaricare e sincronizzare materiale didattico è la stessa. Qui per vostra comodità abbiamo sintetizzata la procedura nel comando (funziona sia per creare che per sincronizzare) 
syncBioTeMo19
Il diario delle lezioni si trova nel file (da aprire all'interno di Jupyter) 
BioTeMo19/lezioni/_DiarioLezioni.ipynb
Questo stesso file contiene anche la bibiografia e l'agenda delle lezioni future.
