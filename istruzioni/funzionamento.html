Il materiale didattico di questo corso è archiviato in formato grezzo in questa repository 
gitlab.com/domenicozambella/BioTeMo19
Per utilizzarlo al meglio, bisogna disporre dell'applicazione Jupyter. Esiste un server Jupyter di ateneo, ma questo stesso server può essere (con un paio di click) installato in qualsiasi computer e usato localmente.

Per utilizzare il server di ateneo bisogna atenticarsi. Le credenziali verranno spedite agli studenti del primo anno per email all'indirizzo istituzionale (a qualcuno sono già state inviate). Per scaricare il materiale didattico bisogna aprire un terminale (dal menu a tendina: New>Terminal) e dare il comando 
syncBioTeMo19
Questa procedura crea la cartella BioTeMo19 con tutto il materiale didattico. La stessa procedura va usata per aggiornare il materiale (che varierà spesso).

Chi installa Jupyter sul proprio computer alla prima sincronizzazione deve usare il comando 
git clone https://gitlab.com/domenicozambella/BioTeMo19.git
In seguito, per aggiornare, deve spostarsi nella directory BioTeMo19 e dare il comando 
git fetch origin; git reset --hard origin/master



Il diario delle lezioni si trova nel file (da aprire all'interno di Jupyter) 
BioTeMo19/lezioni/_DiarioLezioni.ipynb
Questo stesso file contiene anche la bibiografia e l'agenda delle lezioni future.


Per prima cosa quando si apre un notebook (ovvero un file con estensione .ipynb) bisogna eseguire tutte le celle. Dal menù a tendina: Cell>Run All.


Se le cose vanno storte, meglio ri-inizializzare il kernel. Dal menù a tendina: Kernel>Restart. Poi bisogna eseguire di nuovo tutte le celle.
